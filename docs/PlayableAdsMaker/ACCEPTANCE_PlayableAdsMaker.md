# Playable Ads Maker - 验收文档

## 项目概述

Playable Ads Maker是一个用于创建可玩广告的Web应用平台，通过多步骤流程引导用户上传视频、设置互动元素，并最终生成适用于不同广告平台的HTML或ZIP文件。
